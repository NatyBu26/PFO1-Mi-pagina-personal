# Mi página personal - PFO1

Este proyecto es una página personal desarrollada en HTML y CSS como parte de la Práctica Formativa Obligatoria 1 de la materia Desarrollo de Sistemas Web (Front End). En esta página se presentan aspectos de mi perfil personal, intereses, habilidades, películas favoritas, series, canciones y un formulario de contacto, siguiendo la estructura solicitada en la consigna.
